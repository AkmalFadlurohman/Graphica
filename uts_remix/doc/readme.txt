Petunjuk instalasi:
1. Masukan perintah "sudo make" di folder root.
2. Masuk ke direktori "bin".
3. Masukan perintah "sudo ./build" untuk menjalankan aplikasi



Petunjuk penggunaan aplikasi:
1. Setelah aplikasi dijalankan, gerakkan mouse untuk menampilkan menu utama.
2. Gerakkan mouse ke menu yang dipilih, kemudian klik kiri untuk pilih menu.

Petunjuk untuk menu "font":
1. Setelah masuk menu 'font', masukkan karakter yang ingin dicetak di layar, yaitu 'a' - 'z' dan karakter spasi ' '.
2. Untuk keluar dari menu, tekan 'esc'.


Petunjuk untuk menu "Animasi pesawat 1":
1. Setelah masuk menu 'Animasi pesawat 1', animasi langsung ditampilkan. 
2. Untuk keluar dari menu, tekan 'esc'.


Petunjuk untuk menu "Animasi pesawat 2":
1. Setelah masuk menu 'Animasi pesawat 2', animasi langsung ditampilkan. 
2. Untuk keluar dari menu, tekan 'esc'.


Petunjuk untuk menu "Animasi pesawat 3":
1. Setelah masuk menu 'Animasi pesawat 2', untuk control gunakan: 
	- 'w' = gerakkan kamera ke atas;
	- 'a' = gerakkan kamera ke kiri;
 	- 's' = gerakkan kamera ke bawah;
	- 'd' = gerakkan kamera ke kanan;
	- 'z' = kamera zoom in;
	- 'x' = kamera zoom out.
2. Untuk keluar dari menu, tekan 'esc'.


Petunjuk untuk menu "Peta ITB":
1. Setelah masuk menu 'Peta ITB', untuk control gunakan: 
	- 'w' = gerakkan kamera ke atas;
	- 'a' = gerakkan kamera ke kiri;
 	- 's' = gerakkan kamera ke bawah;
	- 'd' = gerakkan kamera ke kanan;
	- '1' = tampilkan/hilangkan gedung;
	- '2' = tampilkan/hilangkan jalan.
2. Untuk keluar dari menu, tekan 'esc'.


Pilih menu 'keluar' untuk keluar dari aplikasi.
